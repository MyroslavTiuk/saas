### Build Project
```
$ yarn  build
```


### To auth Gcloud
```
gcloud auth login
```

### To get Auth list
```
gcloud auth list
```

### Set Account
```
gcloud config set account 'email'
```

### Set Project
```
gcloud config set project elated-fabric-379905
```

gcloud config set compute/region asia-southeast1

gcloud config set compute/zone asia-southeast1-a


### Deploy Project
```
gcloud app deploy
```
